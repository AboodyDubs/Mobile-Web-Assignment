What is jQuery?

jQuery is a collection of pre-written Javascript. This language is intended to ease the use
of Javascript by simplifying multiple statements that are usually entered in Javascript and combines them into a single statement.

Features of jQuery:

  1. Lightweight: it is a light javascript library that is roughly 20KB in size

  2. AJAX: jQuery contains a collection of AJAX commands for simplicity

  3. CSS manipulation: Jquery can interpret CSS and make changes to certain properties

    Example: changing the padding size (Figure 1)

    Figure 1: The jQuery understood the original padding size (which was 10px) and created a function that increases the padding when clicked

  4. DOM manipulation: jQuery uses CSS select methods like Id naming to adjust DOM nodes.

  5. Cross-browser: it is applicable on multiple versions of browser

  6. Deals with creating animations and special effecs for websites

  7. Can manage event handling methods such as hover()
