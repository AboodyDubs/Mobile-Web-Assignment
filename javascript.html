What is Javascript?


A high level Object Oriented Language that allows websites to become interact able with the user. It contains specific interfaces that are able to work with the Document Object Model (DOM), arrays along with the Regular Expression sequences.



Features of Javascript


1- Case sensitive
2- Light weighted language due to its limitation of syntax
3- It is always embedded within websites.
4- Provides the user with opportunities to interact with websites.
5- Can create specific functions that accomplish unique tasks
