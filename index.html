1- Introduction:

Mobile application development is a field of programming that involved the creation of applications for mobile devices. Like desktop applications, the development of mobile phone application involves html layout along with a series of functions ran by a client-side processor such as the Javascript to allow the application interact with the user.


2- Advantages:


- Applications’ accessibility is more frequent considering it can be accessed even outside of houses.
- Easier and simpler navigation for the common user due to the application scaling to the phone’s screen.
- Improves social networking through mobile communications.


3- Disadvantages:


- Sometimes features of an app become compromised due to the small phone screen so that the application can be presented clearly to the common user. (Cannot contain all application features due to limited screen size)
- Slower network (packet transfer not as efficient as computers’)
- Some phones can become incompatible with developed applications.
- Sensitive battery (phone batteries can be consumed faster based on the application’s size)
- Less secure because network packets are always transferred publicly


4- Uses:


- Business trades: e-commerce helps companies make purchases and sales electronically through phone usage.
- Creates time-killing applications (music/game applications that the user can use while waiting for a bus for example.)
- Creates GPS applications (applications that keep track of the users’ location during travel).
