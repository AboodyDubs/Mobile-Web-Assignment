1- What is XML?

XML or Extensible Markup Language is language that organizes specific data so that both the user and the machine can be able to read it. It allows the user to create their own tags rather than be limited to using pre-defined html tags.



2- Features of XML:

  1. Validation: can be validated using the XML schema to check for errors

  2. Compatible with HTML: can be used with operations created through HTML tags

  3. Javascript-compatible: can be made interactive through specific statements written in the script block

  4. Storage efficiency: good for storing document

  5. Extensible: it has extended naming for tags in the sense of allowing users create their own tags

4- Comparing HTML with XML


An XML block of code may be used within an HTML document. However, the two languages differ in certain areas:


  1. HTML does not preserve white space

  2. HTML forces the user to use pre-defined tags, whereas XML allows the user to create their own tags

  3. XML is a dynamic language that carries the information, whereas HTML is static that only displays the data

  4. XML is case sensitive, whereas HTML is not
